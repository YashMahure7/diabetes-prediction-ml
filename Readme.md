# 🧠 Diabetes Prediction using Machine Learning

## 📌 Overview

This project predicts whether a person is diabetic based on medical attributes using multiple machine learning models.

## ⚙️ Tech Stack

* Python
* Pandas
* Scikit-learn
* Matplotlib

## 📊 Models Used

* Logistic Regression
* Decision Tree
* Random Forest
* Support Vector Machine (SVM)

## 📈 Results

Random Forest achieved the highest accuracy among all models.

## 🚀 How to Run

1. Install dependencies:
   pip install -r requirements.txt

2. Run the project:
   python main.py

## 📁 Project Structure

* data/ → dataset
* src/ → core logic
* models/ → saved models
* images/ → visualizations
